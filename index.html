History of the font

David Jonathan Ross is a type designer from Los Angeles, California. In the past he would use adobe to produce his fonts. Now he releases his typefaces through his own company. I chose the variable font LAB DJR. Which was created using on single repeating block. This his first open variations font. Which gives the user the ability to control the roundness, bevel, size, while also being able to split thr pixels. There are 480 characters. There are also 6 variables that the user can choose from BLOCK, PIXEL, GHOST, FLORET, FANCY, DOT. Lastly David created a second part to this font named LAB DJR COLOR. He decided to revisit the original font due to the simplicity and decided to add another more complex aspect. He created a python script that added one of his 8 colors to each pixel.
